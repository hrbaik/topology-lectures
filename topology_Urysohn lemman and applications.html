0. Separation Axioms

Motivation

Not every topological space behaves as nicely as $\mathbb{R}^n$. The separation axioms impose increasingly strong conditions on how well a topology can distinguish points and closed sets using open sets. The theorems in the remaining sections require normality ($T_4$) or regularity ($T_3$), so we begin by setting up these definitions precisely.

0.1   The Hierarchy: $T_1$ through $T_4$

Definition: $T_1$ (Fréchet)

A topological space $X$ is $T_1$ if for every pair of distinct points $x\neq y$, there exists an open set $U$ with $x\in U$ and $y\notin U$.

Proposition: Equivalent characterization of $T_1$

$X$ is $T_1$ if and only if every singleton $\{x\}$ is a closed set.

Proof

($\Rightarrow$) Fix $x\in X$. We show $X\setminus\{x\}$ is open. For any $y\neq x$, the $T_1$ axiom gives an open set $U_y$ with $y\in U_y$ and $x\notin U_y$, i.e. $U_y\subseteq X\setminus\{x\}$. Then $X\setminus\{x\} = \bigcup_{y\neq x} U_y$ is a union of open sets, hence open.

($\Leftarrow$) If every singleton is closed, then given $x\neq y$, the set $U = X\setminus\{y\}$ is open, contains $x$, and does not contain $y$. □

Why $T_1$ alone is weak

$T_1$ guarantees that finite sets are closed, but it says nothing about separating a point from a larger closed set by disjoint neighborhoods. For instance, an infinite set with the cofinite topology is $T_1$ (complements of singletons are cofinite, hence open), but it is not $T_2$: any two nonempty open sets intersect, because their complements are finite while $X$ is infinite.

0.2   $T_2$: Hausdorff Spaces

Definition: $T_2$ (Hausdorff)

A topological space $X$ is $T_2$ (or Hausdorff) if for every pair of distinct points $x\neq y$, there exist disjoint open sets $U,V$ with $x\in U$ and $y\in V$.

Proposition: Uniqueness of limits

In a Hausdorff space, a net (or sequence) converges to at most one point.

Proof

Suppose a net $(x_\alpha)$ converges to both $p$ and $q$ with $p\neq q$. Choose disjoint open sets $U\ni p$ and $V\ni q$. By convergence, $(x_\alpha)$ is eventually in $U$ and eventually in $V$. But then it is eventually in $U\cap V = \varnothing$, a contradiction. □

$T_2$ implies $T_1$

If $x\neq y$, the Hausdorff condition gives disjoint open $U\ni x,\, V\ni y$. In particular, $U$ is an open set containing $x$ but not $y$ (since $U\cap V=\varnothing$). This is exactly the $T_1$ condition. But $T_1$ does not imply $T_2$, as the cofinite topology example above shows.

Key example

Every metric space is Hausdorff. If $x\neq y$ in $(X,d)$, set $\varepsilon = d(x,y) \gt 0$. Then $U = B(x,\varepsilon/2)$ and $V = B(y,\varepsilon/2)$ are disjoint open sets separating $x$ and $y$. (If $z\in U\cap V$, the triangle inequality gives $d(x,y)\le d(x,z)+d(z,y) \lt \varepsilon$, a contradiction.)

0.3   $T_3$: Regular Spaces

Definition: $T_3$ (Regular)

A topological space $X$ is $T_3$ (or regular) if it is $T_1$ and: for every point $x\in X$ and every closed set $C$ with $x\notin C$, there exist disjoint open sets $U,V$ with $x\in U$ and $C\subseteq V$.

Proposition: Equivalent characterization of regularity

A $T_1$ space $X$ is regular if and only if: for every $x\in X$ and every open $W$ containing $x$, there exists an open $U$ with $$x \;\in\; U \;\subseteq\; \overline{U} \;\subseteq\; W.$$

Proof

($\Rightarrow$) Let $x\in W$ with $W$ open. Then $C = X\setminus W$ is closed and $x\notin C$. By regularity, choose disjoint open sets $U\ni x$ and $V\supseteq C$. Since $U\cap V = \varnothing$ we have $U\subseteq X\setminus V$, and $X\setminus V$ is closed, so $\overline{U}\subseteq X\setminus V \subseteq X\setminus C = W$.

($\Leftarrow$) Given $x\notin C$ with $C$ closed, set $W = X\setminus C$. By hypothesis, choose open $U$ with $x\in U\subseteq\overline{U}\subseteq W$. Then $U$ and $X\setminus\overline{U}$ are disjoint open sets with $x\in U$ and $C\subseteq X\setminus\overline{U}$. □

$T_3$ implies $T_2$

If $x\neq y$, then $\{y\}$ is closed (by $T_1$) and $x\notin\{y\}$. Regularity gives disjoint open $U\ni x$ and $V\ni y$. So $X$ is Hausdorff. The converse fails: there exist Hausdorff spaces that are not regular (though such examples are more exotic).

Remark on terminology

Some authors define "regular" without requiring $T_1$, and then write "$T_3$" for "regular $+$ $T_1$." In this course we follow the convention that $T_3$ = regular = $T_1$ + point-closed separation. This is the convention used by Munkres.

0.4   $T_4$: Normal Spaces

Definition: $T_4$ (Normal)

A topological space $X$ is $T_4$ (or normal) if it is $T_1$ and: for every pair of disjoint closed sets $A,B$, there exist disjoint open sets $U,V$ with $A\subseteq U$ and $B\subseteq V$.

$T_4$ implies $T_3$

Given a point $x$ and a closed set $C$ not containing $x$: since $X$ is $T_1$, $\{x\}$ is closed. Now $\{x\}$ and $C$ are disjoint closed sets, so normality gives disjoint open sets $U\supseteq\{x\}$ and $V\supseteq C$. This is exactly the regularity condition.

The full chain

$$T_4 \;\Longrightarrow\; T_3 \;\Longrightarrow\; T_2 \;\Longrightarrow\; T_1$$

Each implication is strict. None reverses in general.

0.5   Sources of Normal Spaces

Theorem: Every metrizable space is normal

Let $(X,d)$ be a metric space and let $A,B\subseteq X$ be disjoint closed sets. Then there exist disjoint open sets separating $A$ and $B$.

Proof

Define $f\colon X\to\mathbb{R}$ by $$f(x) \;=\; \frac{d(x,A)}{d(x,A)+d(x,B)}.$$ This is well-defined since $A\cap B = \varnothing$ and both are closed, so for every $x$, at least one of $d(x,A)$, $d(x,B)$ is positive; in fact $d(x,A)+d(x,B) \gt 0$ for all $x$.

$f$ is continuous (as a ratio of continuous functions with positive denominator), and $f(A) = \{0\}$, $f(B) = \{1\}$. Then $U = f^{-1}\bigl([0,\tfrac{1}{2})\bigr)$ and $V = f^{-1}\bigl((\tfrac{1}{2},1]\bigr)$ are disjoint open sets with $A\subseteq U$ and $B\subseteq V$. □

Theorem: Every compact Hausdorff space is normal

Let $X$ be compact and Hausdorff, and let $A,B\subseteq X$ be disjoint closed sets. Then there exist disjoint open sets separating $A$ and $B$.

Proof

Lemma (point vs. compact closed set). Fix $x\notin B$ where $B$ is closed (hence compact, as a closed subset of a compact space). For each $b\in B$, the Hausdorff condition gives disjoint open sets $U_b\ni x$ and $V_b\ni b$. The collection $\{V_b\}_{b\in B}$ covers $B$; by compactness, finitely many suffice: $B\subseteq V_{b_1}\cup\cdots\cup V_{b_n}$. Set $U_x = U_{b_1}\cap\cdots\cap U_{b_n}$ (open, since it is a finite intersection) and $V_x = V_{b_1}\cup\cdots\cup V_{b_n}$. Then $x\in U_x$, $B\subseteq V_x$, and $U_x\cap V_x = \varnothing$.

Main argument. For each $a\in A$, apply the lemma with $x = a$ to get $U_a\ni a$ and $V_a\supseteq B$ with $U_a\cap V_a = \varnothing$. The collection $\{U_a\}_{a\in A}$ covers $A$; by compactness (of $A$), finitely many suffice: $A\subseteq U_{a_1}\cup\cdots\cup U_{a_m}$. Set $U = U_{a_1}\cup\cdots\cup U_{a_m}$ and $V = V_{a_1}\cap\cdots\cap V_{a_m}$. Then $A\subseteq U$, $B\subseteq V$, and $U\cap V = \varnothing$. □

Summary of key examples

Normal: all metrizable spaces; all compact Hausdorff spaces; all second-countable regular spaces (proved in Section 2 as a step in the Metrization Theorem).

Regular but not normal: the Sorgenfrey plane $\mathbb{R}_\ell\times\mathbb{R}_\ell$. The lower-limit topology on $\mathbb{R}$ is normal, but its product with itself is not.

Hausdorff but not regular: $\mathbb{R}$ with the $K$-topology, where the open sets are generated by intervals $(a,b)$ together with sets $(a,b)\setminus K$ for $K = \{1/n : n\in\mathbb{N}\}$. The closed set $K$ cannot be separated from $0$ by disjoint open sets.

0.6   $T_{3\frac{1}{2}}$: Completely Regular (Tychonoff) Spaces

Definition: $T_{3\frac{1}{2}}$ (Completely Regular)

A topological space $X$ is $T_{3\frac{1}{2}}$ (or completely regular, or a Tychonoff space) if it is $T_1$ and: for every point $x\in X$ and every closed set $C$ with $x\notin C$, there exists a continuous function $f\colon X\to[0,1]$ with $f(x)=0$ and $f(C)=\{1\}$.

Where it fits

$$T_4 \;\Longrightarrow\; T_{3\frac{1}{2}} \;\Longrightarrow\; T_3 \;\Longrightarrow\; T_2 \;\Longrightarrow\; T_1$$

$T_{3\frac{1}{2}} \Rightarrow T_3$: given $f$ with $f(x)=0$ and $f(C)=\{1\}$, the sets $U = f^{-1}([0,\tfrac{1}{2}))$ and $V = f^{-1}((\tfrac{1}{2},1])$ are disjoint open sets separating $x$ from $C$.

$T_4 \Rightarrow T_{3\frac{1}{2}}$: given $x\notin C$ in a normal space, $\{x\}$ and $C$ are disjoint closed sets. Urysohn's Lemma (Section 1) gives a continuous $f\colon X\to[0,1]$ with $f(x)=0$ and $f(C)=\{1\}$. This is one reason Urysohn's Lemma is so fundamental: it establishes that normality implies complete regularity.

The Sorgenfrey plane shows that $T_{3\frac{1}{2}}$ does not imply $T_4$ in general.

0.7   The Shrinking Characterization of Normality

Proposition (Shrinking Lemma)

A $T_1$ space $X$ is normal if and only if: for every closed set $C$ and every open set $W$ with $C\subseteq W$, there exists an open set $V$ with $$C \;\subseteq\; V \;\subseteq\; \overline{V} \;\subseteq\; W.$$

Proof

($\Rightarrow$) Assume $X$ is normal. Let $C$ be closed and $W$ open with $C\subseteq W$. Then $C$ and $X\setminus W$ are disjoint closed sets. By normality, there exist disjoint open sets $V,\,U'$ with $$C \;\subseteq\; V, \qquad X\setminus W \;\subseteq\; U'.$$ Since $V\cap U'=\varnothing$, we have $V\subseteq X\setminus U'$, and $X\setminus U'$ is closed. Therefore $$\overline{V} \;\subseteq\; X\setminus U'.$$ But $X\setminus W\subseteq U'$ means $X\setminus U'\subseteq W$. Combining: $$C \;\subseteq\; V \;\subseteq\; \overline{V} \;\subseteq\; X\setminus U' \;\subseteq\; W.$$

($\Leftarrow$) Let $A,B$ be disjoint closed sets. Set $W = X\setminus B$, which is open with $A\subseteq W$. By hypothesis, there exists an open $V$ with $A\subseteq V\subseteq\overline{V}\subseteq W = X\setminus B$. Then $V$ and $X\setminus\overline{V}$ are disjoint open sets with $A\subseteq V$ and $B\subseteq X\setminus\overline{V}$. □

Why This Matters

The shrinking characterization is the engine behind Urysohn's Lemma (Section 1). To build the continuous function separating two closed sets, we repeatedly "shrink" open sets to produce a nested family $\{U_r\}$ indexed by dyadic rationals. Every single application of normality in that proof goes through this shrinking form. It is also the key tool in the Tietze Extension Theorem (Section 3).

1. Urysohn's Lemma

Recall: Normal Spaces

A topological space $X$ is normal if it is $T_1$ (points are closed) and for every pair of disjoint closed sets $A,B\subseteq X$, there exist disjoint open sets $U,V$ with $A\subseteq U$ and $B\subseteq V$.

Equivalently (and this is the form we use repeatedly), $X$ is normal if and only if: for every closed set $C$ and every open set $W$ with $C\subseteq W$, there exists an open set $V$ with $$C \;\subseteq\; V \;\subseteq\; \overline{V} \;\subseteq\; W.$$ This "shrinking" characterization is the engine of the entire proof.

Theorem 1.1 (Urysohn's Lemma, 1925)

Let $X$ be a normal space and let $A,B\subseteq X$ be disjoint closed sets. Then there exists a continuous function $$f\colon X \to [0,1]$$ such that $f\!\mid_A \equiv 0$ and $f\!\mid_B \equiv 1$.

1.1   Why This Matters

Normal spaces are defined by a purely topological condition—the ability to separate closed sets by open neighborhoods. Urysohn's Lemma upgrades this dramatically: we can separate them by a continuous real-valued function. This passage from topology to analysis is what makes the Lemma the single most important tool in the theory of normal spaces. It underpins both the Metrization Theorem (§2) and the Tietze Extension Theorem (§3), and it is essential for constructing partitions of unity.

1.2   Strategy of the Proof

Key Idea

We want to build $f\colon X\to[0,1]$ with $f^{-1}(0)\supseteq A$ and $f^{-1}(1)\supseteq B$. In a metric space, we could use $f(x) = \frac{d(x,A)}{d(x,A)+d(x,B)}$. But a general normal space has no metric.

Instead, we construct a family of open sets $\{U_r\}$ indexed by the dyadic rationals $$\mathbb{D} \;=\; \Bigl\{\frac{k}{2^n} : k,n\in\mathbb{Z}_{\ge 0}\Bigr\} \;\cap\; [0,1]$$ satisfying the nesting condition: $$r < s \;\;\Longrightarrow\;\; A \;\subseteq\; U_r \;\subseteq\; \overline{U_r} \;\subseteq\; U_s \;\subseteq\; X\setminus B.$$ Then $f(x)=\inf\{r\in\mathbb{D}: x\in U_r\}$ will be the desired function. The density of $\mathbb{D}$ in $[0,1]$ is what makes $f$ continuous.

1.3   Proof — Step 1: Base Case

We begin by constructing $U_0$ and $U_1$.

Step 1 — Base case

Set $U_1 = X\setminus B$. This is open (since $B$ is closed) and $A\subseteq U_1$ (since $A\cap B=\varnothing$).

Now $A$ is closed and $U_1$ is open with $A\subseteq U_1$. By the shrinking characterization of normality, there exists an open set $U_0$ with $$A \;\subseteq\; U_0 \;\subseteq\; \overline{U_0} \;\subseteq\; U_1 = X\setminus B.$$

We now have the chain: $\;A\subseteq U_0 \subseteq \overline{U_0}\subseteq U_1 \subseteq X\setminus B.\;$ In particular $\overline{U_0}\subseteq U_1$, which is the nesting condition for the pair $(r,s)=(0,1)$.

We start with $U_1 = X\setminus B$ (the largest open set avoiding $B$), then use normality to "shrink" it: there exists $U_0$ with $A\subseteq U_0\subseteq\overline{U_0}\subseteq U_1$. 1.4   Proof — Step 2: Inductive Construction on Dyadic Rationals

Step 2 — Induction on dyadic level

Induction hypothesis. Suppose for some $n\ge 0$ we have constructed open sets $U_r$ for every dyadic rational $r = k/2^n \in [0,1]$ such that $$r < s \;\;\Longrightarrow\;\; \overline{U_r}\subseteq U_s. \tag{$\star$}$$

Inductive step. We need to define $U_t$ for each new dyadic $t = (2m+1)/2^{n+1}$, $\;0\le m < 2^n$. Each such $t$ sits between two consecutive old dyadics: $r = m/2^n$ and $s = (m+1)/2^n$, with $t = (r+s)/2$.

By $(\star)$, we have $\overline{U_r}\subseteq U_s$. Since $\overline{U_r}$ is closed and $X\setminus U_s$ is closed, and these are disjoint, normality gives an open set $U_t$ with $$\overline{U_r} \;\subseteq\; U_t \;\subseteq\; \overline{U_t} \;\subseteq\; U_s.$$

Verification. We must check $(\star)$ holds at level $n+1$, i.e., for all dyadics $r' < s'$ at level $\le n+1$, we have $\overline{U_{r'}}\subseteq U_{s'}$. There are three cases:

(a) Both $r',s'$ are old (level $\le n$): $(\star)$ holds by induction.

(b) $r'$ is old, $s' = t$ is new (or $r'$ is new, $s'$ is old): follows from the chain $\overline{U_r}\subseteq U_t\subseteq\overline{U_t}\subseteq U_s$ and transitivity.

(c) Both new: cannot be adjacent at level $n+1$ without an old dyadic between them, so it reduces to (b) by transitivity.

By induction, $U_r$ is defined for all $r\in\mathbb{D}\cap[0,1]$, and $(\star)$ holds globally.

1.5   Proof — Step 3: Defining the Function

Step 3 — Definition of $f$

For each $x\in X$, define $$f(x) \;=\; \inf\bigl\{r\in\mathbb{D}\cap[0,1] : x\in U_r\bigr\}.$$ We adopt the convention $\inf\varnothing = 1$. (But in fact the set is never empty since $U_1 = X\setminus B$ contains every $x\notin B$, and for $x\in B$ we will have $f(x)=1$ by direct verification.)

Claim 1: $f(x)=0$ for all $x\in A$.

Proof. $A\subseteq U_0$, so $0\in\{r: x\in U_r\}$, hence $f(x)\le 0$. Since $f(x)\ge 0$ trivially, $f(x)=0$.

Claim 2: $f(x)=1$ for all $x\in B$.

Proof. For $r < 1$, we have $U_r\subseteq\overline{U_r}\subseteq U_1 = X\setminus B$, so $x\notin U_r$ for any $r < 1$. Thus $\{r: x\in U_r\}\subseteq\{1\}$ (since $x\notin U_1$ either for $x\in B$), actually $x\in B$ means $x\notin U_1$, so $\{r: x\in U_r\} = \varnothing$ and $f(x)=1$.

Claim 3: $0\le f(x)\le 1$ for all $x\in X$.

Proof. The infimum of a subset of $[0,1]$ (or the empty set with convention 1) lies in $[0,1]$.

1.6   Proof — Step 4: Continuity of $f$

Step 4 — Continuity

We show $f$ is continuous by proving that the preimage of every subbasic open set is open. The standard subbasis for $[0,1]$ consists of sets $[0,a)$ and $(a,1]$ for $a\in(0,1)$.

Claim 4: For every $a\in(0,1)$, $$f^{-1}\bigl([0,a)\bigr) \;=\; \bigcup_{\substack{r\in\mathbb{D}\\[2pt] r \lt a}} U_r.$$

Proof.

"$\subseteq$": If $f(x) < a$, then by definition of infimum there exists $r\in\mathbb{D}$ with $r < a$ and $x\in U_r$.

"$\supseteq$": If $x\in U_r$ for some $r < a$, then $f(x)\le r < a$, so $f(x)\in[0,a)$.

The right side is a union of open sets, hence open. ✓

Claim 5: For every $a\in(0,1)$, $$f^{-1}\bigl((a,1]\bigr) \;=\; \bigcup_{\substack{s\in\mathbb{D}\\[2pt] s \gt a}} \bigl(X\setminus\overline{U_s}\bigr).$$

Proof.

"$\supseteq$": If $x\in X\setminus\overline{U_s}$ for some $s > a$, then $x\notin U_s$, so $x\notin U_r$ for any $r\le s$ (by nesting: $U_r\subseteq\overline{U_r}\subseteq U_s$). Hence $f(x)\ge s > a$.

"$\subseteq$": Suppose $f(x) > a$. Pick dyadics $s_1, s_2$ with $a < s_1 < s_2 < f(x)$. Since $f(x) > s_2$, we have $x\notin U_{s_2}$. By nesting, $\overline{U_{s_1}}\subseteq U_{s_2}$, so $x\notin\overline{U_{s_1}}$, i.e., $x\in X\setminus\overline{U_{s_1}}$ with $s_1 > a$.

Again the right side is a union of open sets (each $X\setminus\overline{U_s}$ is open), hence open. ✓

Since $f^{-1}(W)$ is open for every subbasic $W$, and the subbasis generates the topology on $[0,1]$, we conclude $f$ is continuous.

1.7   Remarks

Remarks

1. Converse — characterizing normality. Urysohn's Lemma has a converse: if for every pair of disjoint closed sets in $X$ there exists a continuous $f\colon X\to[0,1]$ separating them, then $X$ is normal. Proof: the sets $f^{-1}([0,1/2))$ and $f^{-1}((1/2,1])$ are disjoint open neighborhoods.

2. Completely regular spaces. A $T_1$ space where points and closed sets can be separated by continuous functions is called completely regular ($T_{3\frac12}$, or Tychonoff). Urysohn's Lemma shows: $T_4 \Rightarrow T_{3\frac12}$. The converse fails (Sorgenfrey plane is $T_{3\frac12}$ but not $T_4$).

3. Why dyadic rationals? The dyadics $\mathbb{D}$ have two crucial properties: (a) density in $[0,1]$, ensuring $f$ is continuous; (b) each new point is the midpoint of two existing points, so only one application of normality is needed per new point. Using all rationals $\mathbb{Q}\cap[0,1]$ would also work (enumerate them and proceed), but dyadics make the induction cleaner.

4. The function is not unique. There are uncountably many Urysohn functions for a given pair $(A,B)$. Different choices of the intermediate open sets yield different functions.

2. Urysohn Metrization Theorem

Recall

A space $X$ is second-countable if it has a countable basis $\mathcal{B}=\{B_n\}_{n=1}^\infty$. It is regular ($T_3$) if it is $T_1$ and for every closed $C$ and $x\notin C$, there exist disjoint open sets separating them.

Fact: Regular + second-countable $\Rightarrow$ normal. (Proof uses the Lindelöf property.)

Theorem 2.1 (Urysohn Metrization Theorem, 1925)

Every regular, second-countable ($T_3$, second-countable) space is metrizable. More precisely, it embeds homeomorphically into the Hilbert cube $$H \;=\; [0,1]^\omega \;=\; \prod_{n=1}^\infty [0,1]$$ equipped with the metric $\;\displaystyle d(\mathbf{x},\mathbf{y}) = \sqrt{\sum_{n=1}^\infty \frac{(x_n-y_n)^2}{n^2}}$.

2.1   Significance

This answers the metrization problem for second-countable spaces: the purely topological conditions of regularity and a countable basis suffice to guarantee a compatible metric. The result covers all second-countable manifolds, all separable Banach spaces (in the norm topology), and all Polish spaces.

2.2   Proof Overview

Strategy

The proof has three main components:

(A) Use the countable basis to produce a countable family of Urysohn functions $\{g_k\}_{k=1}^\infty$ that collectively separate points from closed sets.

(B) Define the embedding $F(x) = (g_1(x), g_2(x), \ldots)$ into $[0,1]^\omega$.

(C) Verify that $F$ is a homeomorphism onto its image. Since $[0,1]^\omega$ is metrizable, so is $F(X)\cong X$.

2.3   Proof — Step 1: Constructing the Separating Family

Step 1 — Countable Urysohn functions

Let $\mathcal{B} = \{B_n\}_{n=1}^\infty$ be a countable basis for $X$. Define the eligible pairs: $$\mathcal{P} \;=\; \bigl\{(n,m)\in\mathbb{N}\times\mathbb{N} : \overline{B_n}\subseteq B_m\bigr\}.$$ Since $\mathcal{P}\subseteq\mathbb{N}\times\mathbb{N}$, it is countable. Enumerate it as $(n_1,m_1),(n_2,m_2),\ldots$

For each $(n_k,m_k)\in\mathcal{P}$: $\overline{B_{n_k}}$ is closed, $X\setminus B_{m_k}$ is closed, and they are disjoint. By Urysohn's Lemma (Theorem 1.1), there exists a continuous function $$g_k\colon X\to[0,1], \quad g_k\big|_{\overline{B_{n_k}}}\equiv 0, \quad g_k\big|_{X\setminus B_{m_k}}\equiv 1.$$

Key property. The family $\{g_k\}$ separates points from closed sets: for any $x\in X$ and any closed $C$ with $x\notin C$, there exists $k$ with $g_k(x)=0$ and $g_k(C)\subseteq\{1\}$.

Why? We apply regularity twice. Since $X\setminus C$ is open and contains $x$:

(i) By regularity, there exists an open set $V$ with $x\in V \subseteq \overline{V}\subseteq X\setminus C$. Since $\mathcal{B}$ is a basis, pick $B_m\in\mathcal{B}$ with $x\in B_m\subseteq V$.

(ii) Now $B_m$ is open and contains $x$. By regularity again, there exists an open set $W$ with $x\in W\subseteq\overline{W}\subseteq B_m$. Pick $B_n\in\mathcal{B}$ with $x\in B_n\subseteq W$. Then $$x \;\in\; B_n \;\subseteq\; \overline{B_n} \;\subseteq\; \overline{W} \;\subseteq\; B_m \;\subseteq\; V \;\subseteq\; X\setminus C.$$ So $(n,m)\in\mathcal{P}$, and the corresponding $g_k$ satisfies $g_k(x)=0$ (since $x\in\overline{B_n}$) and $g_k(c)=1$ for all $c\in C$ (since $C\subseteq X\setminus B_m$).

2.4   Proof — Step 2: The Embedding

Step 2 — Defining $F$ and proving it is injective and continuous

Define $F\colon X\to [0,1]^\omega$ by $$F(x) \;=\; \bigl(g_1(x),\; g_2(x),\; g_3(x),\;\ldots\bigr).$$

$F$ is injective. If $x\neq y$, then $\{y\}$ is closed (by $T_1$) and $x\notin\{y\}$. By the key property, there exists $k$ with $g_k(x)=0$ and $g_k(y)=1$. Hence $F(x)\neq F(y)$.

$F$ is continuous. The product topology on $[0,1]^\omega$ is generated by projections $\pi_k\colon [0,1]^\omega\to[0,1]$. Each composition $\pi_k\circ F = g_k$ is continuous. We claim this implies $F$ is continuous:

Fact: Continuity into a product

Let $Y = \prod_{\alpha} Y_\alpha$ carry the product topology and let $\pi_\alpha\colon Y\to Y_\alpha$ be the projection maps. A map $F\colon X\to Y$ is continuous if and only if $\pi_\alpha\circ F$ is continuous for every $\alpha$.

Proof. ($\Rightarrow$) Each $\pi_\alpha$ is continuous, so compositions are continuous.

($\Leftarrow$) It suffices to check that $F^{-1}(S)$ is open for every subbasis element $S$ of $Y$. The product topology has subbasis $\{\pi_\alpha^{-1}(V) : V\subseteq Y_\alpha \text{ open}\}$. But $$F^{-1}\bigl(\pi_\alpha^{-1}(V)\bigr) \;=\; (\pi_\alpha\circ F)^{-1}(V),$$ which is open in $X$ since $\pi_\alpha\circ F$ is continuous by hypothesis. □

Applying this to our $F = (g_1,g_2,\ldots)$: each $\pi_k\circ F = g_k$ is continuous, so $F$ is continuous.

2.5   Proof — Step 3: $F$ is an Open Map onto $F(X)$

Step 3 — $F$ maps open sets to relatively open sets

It remains to show $F\colon X\to F(X)$ is an open map, which together with continuity and injectivity makes it a homeomorphism.

Let $U\subseteq X$ be open and let $x\in U$. We need to find a neighborhood of $F(x)$ in $F(X)$ contained in $F(U)$.

Since $X\setminus U$ is closed and $x\notin X\setminus U$, the key property gives a $k$ with $g_k(x) = 0$ and $g_k(X\setminus U)\subseteq\{1\}$. Define $$W = \bigl\{z\in F(X) : \pi_k(z) < \tfrac12\bigr\} = F(X)\cap \pi_k^{-1}\bigl([0,\tfrac12)\bigr).$$ This is open in $F(X)$, contains $F(x)$ (since $\pi_k(F(x)) = g_k(x) = 0$), and $W\subseteq F(U)$: if $z = F(y)\in W$, then $g_k(y) < 1/2$, which forces $y\in U$ (since $g_k = 1$ on $X\setminus U$).

Hence $F(U)$ is a neighborhood of each of its points in $F(X)$, so $F(U)$ is open in $F(X)$.

2.6   Proof — Conclusion

Step 4 — Conclusion

$F\colon X\to F(X)\subseteq [0,1]^\omega$ is:

• injective (Step 2),  • continuous (Step 2),  • open onto image (Step 3).

Therefore $F$ is a homeomorphism onto its image. Since $[0,1]^\omega$ is metrizable (by the metric $d$), its subspace $F(X)$ is metrizable, and therefore $X\cong F(X)$ is metrizable.

$\blacksquare$
Remarks

1. Universality of the Hilbert cube. The theorem shows that $[0,1]^\omega$ is a universal space for the class of separable metrizable spaces: every such space embeds into it. This is a strong structural result.

2. More general metrization. Without second-countability, one needs the Nagata–Smirnov Theorem: $X$ is metrizable $\iff$ $X$ is $T_3$ and has a $\sigma$-locally finite basis. Equivalently, by Bing's Theorem, $X$ is metrizable $\iff$ $X$ is $T_3$ and has a $\sigma$-discrete basis.

3. Manifolds. Every topological manifold (Hausdorff, second-countable, locally Euclidean) is metrizable by this theorem, since local Euclideanness implies regularity.

4. Converse direction. Every metrizable space is $T_3$, and every separable metrizable space is second-countable. So the theorem is essentially sharp for separable spaces.

3. Tietze Extension Theorem

Theorem 3.1 (Tietze Extension Theorem)

Let $X$ be a normal space and $A\subseteq X$ a closed subspace.

(a) If $f\colon A\to [a,b]$ is continuous, then there exists a continuous $F\colon X\to[a,b]$ with $F\big|_A = f$.

(b) If $f\colon A\to\mathbb{R}$ is continuous, then there exists a continuous $F\colon X\to\mathbb{R}$ with $F\big|_A = f$.

3.1   Significance

The Tietze Extension Theorem is among the most widely applied results in all of topology. It tells us that continuous functions defined on closed subsets always extend, with no increase in range. This is essential for constructing partitions of unity, extending homotopies (the Homotopy Extension Property), building retractions, and in functional analysis for extending operators. The theorem also characterizes normality: $X$ is normal if and only if it has the Tietze extension property.

3.2   Proof Strategy

Key Idea — The "Thirds Trick" Iterated

We prove part (a) for $f\colon A\to[-1,1]$ (the general bounded case $[a,b]$ follows by rescaling). The idea is to build $F = \sum_{n=1}^\infty g_n$ where each $g_n\colon X\to\mathbb{R}$ is constructed by Urysohn's Lemma. At each step, $g_n$ is designed to reduce the maximum error $\|f-\sum_{k=1}^n g_k\|_A$ by a factor of $2/3$.

Specifically, given a function $h\colon A\to[-M,M]$, we partition the range into three equal intervals and use Urysohn's Lemma on the two "extreme" preimages to construct $g\colon X\to[-M/3, M/3]$ that approximates $h$ to within $2M/3$ on $A$. This is the "thirds trick."

3.3   Proof — The Core Lemma (Thirds Trick)

Lemma 3.2 (Thirds Trick)

Let $X$ be normal, $A\subseteq X$ closed, and $h\colon A\to[-M,M]$ continuous. Then there exists a continuous $g\colon X\to[-M/3, M/3]$ such that $$\bigl|h(x) - g(x)\bigr| \;\le\; \frac{2M}{3} \quad\text{for all } x\in A.$$

Proof. Define $$A^- = h^{-1}\!\bigl([-M, -M/3]\bigr), \qquad A^+ = h^{-1}\!\bigl([M/3, M]\bigr).$$ Both are closed in $A$ (preimages of closed sets under $h$), hence closed in $X$ (since $A$ is closed in $X$). They are disjoint.

By Urysohn's Lemma, there exists a continuous $g\colon X\to[-M/3, M/3]$ with $g(A^-)=\{-M/3\}$ and $g(A^+)=\{M/3\}$.

Error estimate on $A$: For $x\in A$, there are three cases:

Case 1: $x\in A^-$, i.e., $h(x)\in[-M,-M/3]$. Then $g(x)=-M/3$, so $h(x)-g(x)\in[-M+M/3,\, 0]=[-2M/3,\, 0]$.

Case 2: $x\in A^+$, i.e., $h(x)\in[M/3,M]$. Then $g(x)=M/3$, so $h(x)-g(x)\in[0,\, 2M/3]$.

Case 3: $h(x)\in[-M/3,M/3]$ and $g(x)\in[-M/3,M/3]$. So $|h(x)-g(x)|\le 2M/3$.

In all cases, $|h(x)-g(x)|\le 2M/3$.

Step 2 ($A^-$ and $A^+$): The preimages $A^-=h^{-1}([-M,-M/3])$ and $A^+=h^{-1}([M/3,M])$ are identified as disjoint closed sets (highlighted on the curve and on the domain axis). Step 3 (Urysohn): Urysohn's Lemma produces $g\colon X\to[-M/3,M/3]$ with $g(A^-)=-M/3$ and $g(A^+)=M/3$. Step 4 (Residual): The difference $h-g$ on $A$ lies in $[-2M/3,\, 2M/3]$ — the range has shrunk by factor $2/3$. This factor is the engine of convergence.

3.4   Proof — Iterating the Thirds Trick

Step 1 — Building the series

Apply Lemma 3.2 iteratively. Start with $h_0 = f\colon A\to[-1,1]$ (so $M_0=1$).

Step $n=1$: Apply Lemma 3.2 to $h_0$ with $M=1$. Get $g_1\colon X\to[-1/3,1/3]$ with $\|f-g_1\|_A\le 2/3$.

Step $n=2$: Define $h_1 = f - g_1\big|_A\colon A\to[-2/3, 2/3]$. Apply Lemma 3.2 to $h_1$ with $M=2/3$. Get $g_2\colon X\to[-\frac{1}{3}\cdot\frac{2}{3},\, \frac{1}{3}\cdot\frac{2}{3}]$ with $\|h_1 - g_2\|_A \le (2/3)^2$, i.e., $\|f - g_1 - g_2\|_A\le(2/3)^2$.

Step $n$ (general): We have $h_{n-1} = f - \sum_{k=1}^{n-1}g_k\big|_A\colon A\to[-(2/3)^{n-1},\, (2/3)^{n-1}]$. Apply Lemma 3.2 with $M = (2/3)^{n-1}$. Get $$g_n\colon X\to\Bigl[-\frac{1}{3}\Bigl(\frac{2}{3}\Bigr)^{n-1},\; \frac{1}{3}\Bigl(\frac{2}{3}\Bigr)^{n-1}\Bigr]$$ with $$\Bigl\|f - \sum_{k=1}^n g_k\Bigr\|_A \;\le\; \Bigl(\frac{2}{3}\Bigr)^n. \tag{$\dagger$}$$

3.5   Proof — Convergence and Properties of $F$

Step 2 — Uniform convergence of $\sum g_n$ on $X$

Write $S_n = \sum_{k=1}^n g_k$ for the partial sums and define $F(x) = \sum_{n=1}^\infty g_n(x)$. Since $\|g_n\|_\infty \le \frac{1}{3}\bigl(\frac{2}{3}\bigr)^{n-1}$, the tail of the series satisfies: for every $x\in X$, $$|F(x) - S_n(x)| \;=\; \Bigl|\sum_{k=n+1}^\infty g_k(x)\Bigr| \;\le\; \sum_{k=n+1}^\infty \frac{1}{3}\Bigl(\frac{2}{3}\Bigr)^{k-1} \;=\; \Bigl(\frac{2}{3}\Bigr)^n \;\xrightarrow{n\to\infty}\; 0.$$ This bound is independent of $x$, so $S_n\to F$ uniformly on $X$.

Step 3 — $F$ is continuous and extends $f$

Continuity. Each $g_n$ is continuous (constructed via Urysohn's Lemma). A uniform limit of continuous functions on a topological space is continuous. Hence $F$ is continuous.

$F$ extends $f$. For $x\in A$, by ($\dagger$), $|f(x) - S_n(x)| \le (2/3)^n \to 0$, so $S_n(x) \to f(x)$. But $S_n \to F$ pointwise (in fact uniformly), so $F(x) = f(x)$ for all $x\in A$.

$F$ maps into $[-1,1]$. For all $x\in X$: $$|F(x)| \;\le\; \sum_{n=1}^\infty |g_n(x)| \;\le\; \sum_{n=1}^\infty \frac{1}{3}\Bigl(\frac{2}{3}\Bigr)^{n-1} = 1.$$

3.6   Proof — Extending to the Unbounded Case

Step 4 — Part (b): $f\colon A\to\mathbb{R}$

Let $\varphi\colon\mathbb{R}\xrightarrow{\;\sim\;}(-1,1)$ be a homeomorphism, e.g., $\varphi(t) = t/(1+|t|)$. Then $\varphi\circ f\colon A\to (-1,1)\subseteq [-1,1]$ is continuous.

By part (a), extend to $G\colon X\to[-1,1]$ with $G\big|_A = \varphi\circ f$.

Problem: $G$ might attain the values $\pm 1$, where $\varphi^{-1}$ is undefined. Define $$C = G^{-1}\bigl(\{-1\}\cup\{1\}\bigr).$$ $C$ is closed in $X$ (preimage of a closed set). Moreover $C\cap A = \varnothing$, since $G(A) = \varphi(f(A))\subseteq(-1,1)$.

Since $A$ is closed and $C$ is closed with $A\cap C = \varnothing$, Urysohn's Lemma gives $\lambda\colon X\to[0,1]$ with $\lambda\big|_A\equiv 1$ and $\lambda\big|_C\equiv 0$.

Define $\widetilde{G}(x) = \lambda(x)\cdot G(x)$. Then:

• $\widetilde{G}$ is continuous (product of continuous functions).

• $\widetilde{G}\big|_A = 1\cdot G\big|_A = \varphi\circ f$.

• $\widetilde{G}(X)\subseteq (-1,1)$: if $G(x) = \pm1$ then $x\in C$ so $\lambda(x) = 0$ and $\widetilde{G}(x) = 0$. Otherwise $|G(x)| < 1$ so $|\widetilde{G}(x)|\le|G(x)| < 1$.

Finally, $F = \varphi^{-1}\circ\widetilde{G}\colon X\to\mathbb{R}$ is continuous and $F\big|_A = \varphi^{-1}\circ\varphi\circ f = f$.

$\blacksquare$
Remarks

1. Converse — characterization of normality. If $X$ has the Tietze extension property (every continuous $f\colon A\to[0,1]$ on every closed $A$ extends to $X$), then $X$ is normal. Proof: for disjoint closed $A,B$, the function $f(x) = 0$ on $A$, $f(x) = 1$ on $B$ is continuous on the closed set $A\cup B$; any extension is a Urysohn function.

2. Necessity of closedness. If $A$ is not closed, extension may fail. Example: $f(x) = \sin(1/x)$ on $(0,1]\subseteq\mathbb{R}$ has no continuous extension to $[0,1]$.

3. The Dugundji Extension Theorem (1951). For metric spaces, a much stronger result holds: if $A$ is closed in a metric space $X$ and $E$ is a locally convex TVS, then every continuous $f\colon A\to E$ extends to $F\colon X\to E$ with $F(X)\subseteq\operatorname{conv}(f(A))$. The extension is even a linear operator $C(A,E)\to C(X,E)$.

4. Applications.

• Partitions of unity on paracompact Hausdorff spaces use Tietze/Urysohn type extensions.

• CW-complex theory: the Homotopy Extension Property for CW-pairs relies on Tietze-type arguments.

• Dimension theory: Lebesgue covering dimension can be characterized using extension properties (Eilenberg–Otto).
